# Car Marketplace — Ethiopia & East Africa

A mobile-first car marketplace for Ethiopia and East Africa, built with React, Vite, and Tailwind CSS.

## Features

- 8 East African countries supported (Ethiopia, Kenya, Uganda, Tanzania, Rwanda, Djibouti, Somalia, South Sudan)
- Hierarchical location picker (Country → Region → City → Area)
- Multi-currency (ETB, KES, UGX, TZS, RWF, DJF, SOS, SSP)
- License plate code filtering
- Duty status filtering (Duty paid / Duty free / Customs cleared)
- Telegram contact integration
- Optional GPS pin for sellers
- In-app messaging between buyers and sellers
- Dealer verification (At Dealer tab)
- Photo upload for listings
- Buyer/seller authentication

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 4
- Lucide React (icons)

## Development

This project is deployed via Vercel and connected to GitHub for automatic deployment on push.
